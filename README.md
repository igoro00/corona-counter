# corona-counter
Arduino project who download and display number of infected by covid-19
# how to use
1.Download script named corona and run it in terminal using command sh /path/to/script if you not want information from Polish you should edit corona script eg. if you want to get number of infected people from world you must edit https://corona.lmao.ninja/countries/Poland to https://corona.lmao.ninja/all

2.Download arduino code named corona.ino and flash it to arduino using arduino ide or vscode but if you edit script you also should edit arduino code eg. edit text in poland to in world
